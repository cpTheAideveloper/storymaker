## App Title: My Story App

**Description:** A web application that generates personalized stories based on user-selected genres, narrative styles, and audience.

**Structure:**
```
src/
├── app/
│   ├── api/
│   │   └── generate-story/
│   │       └── route.ts
│   └── page.tsx
├── components/
│   ├── ui/
│   │   ├── Onboarding/
│   │   │   ├── GenreStep.tsx
│   │   │   ├── FormatStep.tsx
│   │   │   ├── AudienceStep.tsx
│   │   │   └── IdeaStep.tsx
│   │   ├── StoryDisplay.tsx
│   │   └── Loading.tsx
├── types/
│   └── index.d.ts
└── utils/
    └── api.ts
```

**Technology:**
- TypeScript
- Next.js
- OpenAI API
